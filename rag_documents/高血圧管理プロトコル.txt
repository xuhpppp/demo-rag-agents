高血圧管理プロトコル
====================
出典：日本高血圧学会ガイドライン（参考資料）
改訂版：2024年

■ 1. 高血圧の定義と分類

高血圧とは、診察室血圧が収縮期 140 mmHg 以上または拡張期 90 mmHg 以上の状態をいう。

【血圧分類（診察室血圧）】
| 分類 | 収縮期血圧 | | 拡張期血圧 |
|------|-----------|---|-----------|
| 正常血圧 | < 120 | かつ | < 80 |
| 正常高値血圧 | 120〜129 | かつ | < 80 |
| 高値血圧 | 130〜139 | または | 80〜89 |
| I度高血圧 | 140〜159 | または | 90〜99 |
| II度高血圧 | 160〜179 | または | 100〜109 |
| III度高血圧 | ≥ 180 | または | ≥ 110 |
| 孤立性収縮期高血圧 | ≥ 140 | かつ | < 90 |

【原発性 vs 二次性】
- 原発性（本態性）高血圧：原因不明・全体の 90〜95%
- 二次性高血圧：原因疾患あり（腎実質性、腎血管性、原発性アルドステロン症、甲状腺疾患など）

■ 2. 高血圧による臓器障害リスク

【心臓】
- 左室肥大、心不全、冠動脈疾患（心筋梗塞）
- 心房細動のリスク増加

【脳】
- 脳卒中（出血性・虚血性）
- 認知症のリスク増加

【腎臓】
- 慢性腎臓病（CKD）の進行促進
- 尿タンパクの出現

【眼】
- 高血圧性網膜症

■ 3. 治療目標血圧

| 患者群 | 診察室血圧目標 |
|--------|--------------|
| 75歳未満の成人 | < 130/80 mmHg |
| 75歳以上の高齢者 | < 140/90 mmHg |
| 糖尿病合併 | < 130/80 mmHg |
| CKD合併（蛋白尿あり） | < 130/80 mmHg |
| 心不全合併 | < 130/80 mmHg |
| 脳卒中既往 | < 130/80 mmHg |

■ 4. 薬物療法

【第一選択薬（5大降圧薬）】

1. ACE阻害薬（例：リシノプリル Lisinopril、エナラプリル Enalapril）
   - 適応：糖尿病性腎症、心不全、心筋梗塞後
   - 禁忌：妊婦、両側腎動脈狭窄、高カリウム血症
   - 副作用：空咳（10〜20%）、血管浮腫

2. ARB（アンジオテンシンII受容体拮抗薬）（例：ロサルタン Losartan、バルサルタン）
   - ACE阻害薬に準じた適応、空咳が少ない
   - 禁忌：妊婦（ACE阻害薬と同様）

3. カルシウム拮抗薬（例：アムロジピン Amlodipine、ニフェジピン）
   - 高齢者・狭心症・脳卒中既往に有効
   - 副作用：浮腫（足首）、顔面紅潮、頭痛

4. サイアザイド系利尿薬（例：ヒドロクロロチアジド Hydrochlorothiazide）
   - 骨粗鬆症予防効果、心不全に有効
   - 副作用：低カリウム血症、高尿酸血症、高血糖

5. β遮断薬（例：アテノロール Atenolol、カルベジロール Carvedilol）
   - 適応：狭心症、頻脈性不整脈、心不全
   - 禁忌：気管支喘息、徐脈、房室ブロック

【主要薬剤と対応するSNOMED-CTコード（参考）】
- リシノプリル（Lisinopril）：386873009
- アムロジピン（Amlodipine）：386864001
- ヒドロクロロチアジド（Hydrochlorothiazide）：387525002
- アテノロール（Atenolol）：372802000

【多剤併用療法】
- 2剤以上の組み合わせ例：
  - ACE阻害薬/ARB + カルシウム拮抗薬
  - ACE阻害薬/ARB + サイアザイド系利尿薬
  - カルシウム拮抗薬 + サイアザイド系利尿薬
- 3剤難治性高血圧（RH）：上記3剤でもコントロール不良の場合 → ミネラルコルチコイド受容体拮抗薬（スピロノラクトン）を追加

■ 5. 非薬物療法（生活習慣の改善）

降圧効果を最大化するため、薬物療法と併用する。

| 介入 | 期待降圧効果（収縮期） |
|------|----------------------|
| 減塩（< 6g/日） | −2〜8 mmHg |
| 体重減少（10kg） | −5〜20 mmHg |
| 有酸素運動（週5日 30分） | −4〜9 mmHg |
| 禁煙 | 間接効果（CV リスク低下） |
| 節酒（男性 < 20g/日、女性 < 10g/日） | −2〜4 mmHg |
| DASH食（野菜・果物・低脂肪乳製品中心） | −8〜14 mmHg |

■ 6. モニタリングと追跡

【家庭血圧測定の推奨】
- 朝（起床後1時間以内、排尿後、服薬前）：2回測定・平均値を記録
- 晩（就寝前）：2回測定・平均値を記録
- 家庭血圧目標：< 125/75 mmHg

【追跡スケジュール】
| 状況 | 受診間隔 |
|------|---------|
| 新規処方後 | 1〜4週間後 |
| 血圧安定後 | 3〜6ヶ月ごと |
| 臓器障害あり | 1〜3ヶ月ごと |

【検査項目】
- 血液検査：血清クレアチニン、電解質（Na, K）、血糖、脂質
- 尿検査：タンパク尿、微量アルブミン
- 心電図：左室肥大の評価
- 眼底検査：高血圧性網膜症の評価

■ 7. 高血圧緊急症・切迫症

【高血圧緊急症（Emergency）】
- 定義：重篤な臓器障害を伴う血圧上昇（通常 > 180/120 mmHg）
- 対処：速やかな降圧（最初の1時間で 20〜25% 低下）
- 使用薬：静注ニカルジピン、ラベタロール

【高血圧切迫症（Urgency）】
- 定義：臓器障害なし、血圧 > 180/120 mmHg
- 対処：経口薬で数時間〜24時間かけて緩徐に降圧
