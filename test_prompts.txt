2. Aggregation

5. Business logic

# RAG Agent (guidelines only)
# (RAGドキュメント: rag_documents/ フォルダ内の4つのガイドライン文書を参照)

1. アレルギー — アナフィラキシー対応
アナフィラキシーの診断基準と、発症時のエピネフリン投与量（成人・小児）を含む緊急対応手順を教えてください。

2. 医療保険 — HMOとPPOの違い
HMOとPPOの保険プランの違い（ネットワーク制限・紹介状の要否・自己負担額）を比較して説明してください。

3. 糖尿病 — 合併症モニタリング
糖尿病の微小血管合併症（網膜症・腎症・神経障害）の定期検査項目と推奨される検査頻度を教えてください。

4. 高血圧 — 非薬物療法
高血圧の非薬物療法（減塩・運動・DASH食・体重管理）について、それぞれの期待される降圧効果を教えてください。

5. アレルギー — ペニシリンの交差反応
ペニシリンアレルギー患者におけるセファロスポリン系抗生物質との交差反応率と、代替薬の選択方針を教えてください。

6. 医療保険 — 請求拒否と不服申立て
医療保険の請求が拒否される主な理由と、不服申立て（アピール）の手順を教えてください。

# Combine RAG and SQL Agent
# (RAGドキュメント: rag_documents/ フォルダ内の4つのガイドライン文書を参照)

1. 糖尿病 — ガイドライン + 患者数 - ???
糖尿病のHbA1c管理目標と推奨される第一選択薬をガイドラインから教えてください。また、データベース内で現在糖尿病（Diabetes）の診断を持つ患者は何人いますか？

2. 高血圧 — 推奨薬 vs 実際の処方
高血圧治療ガイドラインが推奨する第一選択薬の種類を挙げてください。その上で、データベース内の高血圧（Hypertension）患者に実際に処方されている薬の上位5種類と処方件数を表示してください。

3. ペニシリンアレルギー — 管理指針 + 患者一覧
ペニシリンアレルギーの症状・交差反応・対処法についてガイドラインの内容を要約してください。また、データベース内でペニシリン（Penicillin）アレルギーを登録している患者の一覧（氏名・アレルギーの種類）を表示してください。

4. 保険カバー率 — 制度説明 + 実績データ
メディケア（Medicare）とメディケイド（Medicaid）の保険カバー率の違いについてガイドラインの説明をまとめてください。次に、データベース上でメディケアとメディケイドそれぞれの平均カバー率（PAYER_COVERAGE / TOTAL_CLAIM_COST）を計算して比較してください。

5. 予防接種 — 推奨スケジュール + 接種実績
成人向けの主要ワクチン（インフルエンザ・肺炎球菌・帯状疱疹）の推奨接種スケジュールをガイドラインから説明してください。データベース内で最も多く記録されている免疫接種（Immunization）の上位10種類と件数も合わせて表示してください。
